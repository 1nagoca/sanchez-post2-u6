# 📦 Sistema de Inventario de Productos (MVC con JSP y Servlets)

Este proyecto es una aplicación web desarrollada en Java que implementa el patrón de arquitectura **MVC (Modelo-Vista-Controlador)**. Permite gestionar un catálogo de productos realizando operaciones CRUD (Crear, Leer, Actualizar y Eliminar) utilizando memoria temporal.

Proyecto correspondiente a la **Unidad 6: JSP con MVC (Post-Contenido 1)** de la Universidad Francisco de Paula Santander (UFPS).

## 🚀 Funcionalidades Implementadas

* **Listado de Productos:** Vista principal que muestra el inventario actual.
* **Registro de Nuevos Productos:** Formulario validado para añadir ítems al catálogo.
* **Edición de Productos:** Carga dinámica de los datos de un producto existente para su modificación.
* **Eliminación:** Borrado de productos del inventario con confirmación del usuario.
* **Patrón PRG (Post-Redirect-Get):** Implementado para evitar el reenvío duplicado de formularios al recargar la página.

## 🛠️ Tecnologías Utilizadas
* **Backend:** Java, Jakarta EE (Servlets)
* **Frontend:** JSP, JSTL, Expression Language (EL), HTML5, CSS3
* **Servidor Web:** Apache Tomcat
* **Gestor de Dependencias:** Maven

## ⚙️ Prerrequisitos

Para ejecutar este proyecto en tu entorno local, necesitas:
1. **Java Development Kit (JDK):** Versión 11 o superior.
2. **Apache Tomcat:** Versión 10.x (compatible con `jakarta.servlet`).
3. **IDE:** IntelliJ IDEA (recomendado) o Eclipse.
4. **Git:** Para clonar el repositorio.
